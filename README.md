
## PROYECTO LAVANDERIA EN SPRING BOOT V1.0.1

- Actualizacion de Backend de mi primer proyecto de lavanderia anteriormente creada con NodeJs y Express con esta nueva actualizacion implementamos su seguridad con autenticacion con Json Web Token y autorizacion por roles donde empleados de cada area podra solo modificar el estado de el paquete ingresado con el cliente podra tener una nueva opcion de ver en tiempo real su estado de paquete todo esto dentro de una base de datos en MYSQL.
